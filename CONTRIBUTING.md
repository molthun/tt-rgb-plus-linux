# Contributing

Hardware support depends on real USB HID controllers, so useful reports should
include:

- controller model
- `lsusb` line for the controller
- `tt-rgb-plus list --all`
- `tt-rgb-plus sensors`
- what commands were tested
- whether fan speed, RGB color, and RGB effects worked

Please do not commit captured proprietary Thermaltake binaries or installers.
Small protocol notes, command bytes, and test results are welcome.
